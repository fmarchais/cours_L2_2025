Introduction à la gestion des données avec R

Titre page

Pour programmer en R, on utilise généralement un Environnement de développement, ou IDE, un logiciel qui permet d’écrire du code R et de l’exéctuer, comme RStudio.

L’écran de RStudio est découpé en quatre parties

  • En haut à gauche : le script (l’entièreté du code, que l’on pourra enregistrer pour le réutiliser)

  • En haut à droite : l’environnement (indique les données enregistrées)

  • En bas à gauche : la console (exécute le code)

  • En bas à droite : le Viewer : permet de visualiser le résultat du code

Pour ce cours, vous utiliserez les blocs de code interactifs du document. Grâce à un outil appelé WebR, le code R est traduit en WebAssembly, un langage utilisé par les navigateurs Web.
